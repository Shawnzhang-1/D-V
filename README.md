# 数据可视化应用

一个基于 React + Recharts 的数据可视化工具，支持 CSV 和 Excel 文件上传，提供多种图表类型和个性化配置。

## 功能特性

- 📁 **文件支持**: CSV、Excel (.xlsx/.xls) 文件上传
- 📊 **多种图表**: 折线图、柱状图、饼图、散点图、面积图、混合图
- 🎨 **个性化配置**: 自定义颜色、标题、图例、网格等
- 📈 **数据范围选择**: 自由选择显示的数据范围
- 💾 **导出功能**: 导出图表为 PNG，导出数据为 CSV
- 🚀 **高性能**: 大文件缓存、智能采样

## 在线使用

👉 [点击打开应用](https://你的用户名.github.io/data-visualization-app/)

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- React 19
- TypeScript
- Vite
- Recharts
- Tailwind CSS
- Papa Parse (CSV)
- xlsx (Excel)
